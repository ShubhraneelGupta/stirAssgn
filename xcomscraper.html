X.COM SCRAPING ASSIGNMENT (https://react-shenanigans-nine.vercel.app/)

  • The front-end - https://react-shenanigans-nine.vercel.app/

    • github link - https://github.com/ShubhraneelGupta/ReactShenanigans/blob/main/src/App.jsx
    • Written in react and built using vite, you need to click on fetch and will have to wait for about 20-25 secs

  • The back-end - https://patient-firefly-fun.ngrok-free.app/

    • github link - https://github.com/ShubhraneelGupta/xComScraper
    • This runs a headless instance of chromium with selenium within a docker container

    • Caveats

    • It’s hosted locally - yeah I tried railway.com and render.com but their free tiers are so slow that I had to do cumulative explicit waits of over a minute so I decided to use NGROK and forward my port which runs the docker container so it can be accessed from the web. But I do need to keep my laptop connected to the internet all the time.

    • API

    • /api/my-ip —GET
      • Returns the ip that was used to make the request to x.com
    • /api/save —POST
      • Saves the data that has been scraped into a mongoDB cluster
    • /api/scrape —GET
      • Heart of the API, navigates Login and through other elements to reach to the top 5 trending topics on x.com

    • Stack

    • Flask - for the server side logic and routing
    • Docker - for containerizing
    • Selenium - for Scraping
    • MongoDB - for saving the fetched data
    • Proxymesh - for IP rotation (commented out)

  • Might fail -

    • Sometimes because of logging in too many times, they ask different things thus changing the layout and I have not figured out how to deal with it without taking even more processing time but if that happens pls contact me - I will change the account name and password and it will give me the good ol’ format that my script is written for .

  • What I failed to implement -

    • Proxymesh - Given I can only use one proxy from their list and they rotate quite slowly I kept on getting banned, so I commented it out, Still figuring out a way to use it.
